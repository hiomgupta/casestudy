Core Features

Two workflows. Hundreds of use cases. Human in control at every step.

Feature A

Magic Quill

Sketch roughly. Type your intent. Done.

Qwen Image Edit · SUPIR · MobileSAM
  • 1Sketch a rough region on the canvas
  • 2Type what you want in plain language
  • 3AI interprets stroke + prompt together
  • 4Diff view — accept, branch, or retry
Feature B

Smart Prompt Panel

Tap a preset. Or just type. No menus required.

Qwen Relight · JuggernautXL · CLIP
  • 1Tap a preset or type a custom prompt
  • 2AI auto-detects regions and subjects
  • 3On-device inference runs first (private)
  • 4Cloud handles heavy edits in <10 seconds

Design Rationale

Every decision traces to research, hardware constraints, or user psychology.

01

Glassmorphism for the wearable era

Translucent surfaces keep focus on the canvas, not the chrome — built for what's next.

02

Hick's Law — decide faster

Tools grouped into families, progressively revealed. Casual users get simplicity; pros get depth.

03

Thumb-optimised ergonomics

Every primary action lives within natural thumb reach. No reaching. Ever.

04

Human stays the pilot

Diff view lets users compare, branch, accept or reject. AI suggests — creator decides.

05

Multimodal by default

Voice, stylus, gesture — first-class modes for the next generation of mobile creators.

06

Visual search & retrieval

Search by what's inside an image. The photo library becomes a semantic creative database.

Ethics & Build

Safety is architecture, not afterthought. And this is a working prototype — not a concept deck.

🛡️

Prompt Safety Layer

Blocks explicit content, unsafe depictions, harassment. Clear educational refusals — not cryptic errors.

🔖

AI Edit Disclosure

Every AI output marked with a visible watermark, aligned with C2PA provenance standards.

📋

Transparent Models

All models, licenses, and inference choices fully documented. No black boxes.

🔐

On-Device Privacy

Selection and segmentation happen on-device via NPU. Personal images never leave for lightweight ops.
